# Santhoshkumar G - Portfolio

A modern, high-performance portfolio website built with **Next.js 16**, **React 19**, and **Tailwind CSS 4**. This project showcases my skills, projects, and experience with a focus on premium design, smooth animations, and interactivity.

## 🚀 Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations:** 
  - [Framer Motion](https://www.framer.com/motion/) (Page transitions & element animations)
  - [GSAP](https://gsap.com/) (High-performance complex animations)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Utilities:** `clsx`, `tailwind-merge`

## ✨ Features

- **Dynamic Background:** Custom `DotGrid` component with interactive "spotlight" effect using GSAP.
- **Responsive Design:** Fully responsive layout optimized for all devices.
- **Smooth Animations:** Scroll-triggered animations and hover effects using Framer Motion.
- **Modern UI:** Glassmorphism, gradients, and a sleek dark theme.
- **Sections:**
  - **Hero:** Impactful introduction with gradient text and call-to-action.
  - **About:** Professional summary.
  - **Skills:** Visual representation of technical expertise.
  - **Projects:** Showcase of key projects with links to demos and code.
  - **Experience:** Timeline of professional journey.
  - **Contact:** Interactive contact form and social links.

## 🛠️ Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/santhoshkumar555/Portfolio.git
    cd Portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Navigate to [http://localhost:3000](http://localhost:3000) to view the site.

## 📦 Build for Production

To create an optimized production build:

```bash
npm run build
```
